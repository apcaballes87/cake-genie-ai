# Plan: Customize Add to Cart Bar - Icon Only for 420px-639px

## Objective
In the customizing page's "add to cart bar", from breakpoint 420px to 639px, change the share and chat buttons to show only icons (remove text).

## Technical Details
- **Tailwind CSS v4** is used in this project
- Tailwind v4 supports arbitrary values with bracket syntax: `min-[420px]:max-[639px]:hidden`
- Current responsive classes use `sm:` breakpoint (640px), need to add custom range 420px-639px

## Files to Modify

### 1. `/Users/apcaballes/genieph-nextjs/src/components/ShareButton.tsx`

Two components need updating:
- `ShareButton` (lines 14-64)
- `ChatButton` (lines 74-115)

### Current Classes:
```tsx
// Button className
className={`
  flex items-center justify-center
  w-11 h-11 sm:w-auto sm:h-12 sm:px-4 sm:gap-2
  ...
`}
```

### Required Changes:

1. **Update button className** - Add custom range for 420px-639px to remove padding when text is hidden:
   - Change `w-11 h-11 sm:w-auto sm:h-12 sm:px-4 sm:gap-2` 
   - To `w-11 h-11 sm:w-auto sm:h-12 min-[420px]:max-[639px]:px-0 min-[420px]:max-[639px]:gap-0 sm:px-4 sm:gap-2`

2. **Update text spans** - Hide text in 420px-639px range:
   - Change `{showText && <span>Share</span>}` to `<span className="min-[420px]:max-[639px]:hidden">Share</span>`
   - Apply same change for "Sharing...", "Chat", "Connecting..."
